Why a Small Language Model?

SLMs offer a perfect balance of intelligence and efficiency for mobile devices.

Privacy First Design

  • Android UsageStatsManager: We use standard Android APIs to read usage data securely.
  • Local Processing: All analysis happens right on your device's processor.
  • No Login Required: Just install and use. No accounts, no passwords.
  • No Cloud APIs: We don't send API requests to OpenAI, Google, or anyone else.
  • No Third-Party Analytics: We don't track how you use the app.
